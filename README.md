Module-
Ring Buffer (Circular Buffer)

Description-
This project implements a fixed-size circular buffer of 8 bytes using C.

Features-
1-Buffer initialization
2-Write operation
3-Read operation
4-Buffer count query
5-Full buffer detection
6-Empty buffer detection
7-FIFO (First In First Out) data handling
8-Circular wrap-around using bitwise operations Files
